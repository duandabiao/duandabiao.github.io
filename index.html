PHP

#一、PHP的简介

PHP是一种创建动态交互性站点的强有力的服务器端脚本语言。

什么是php

  • PHP 是 “PHP Hypertext Preprocessor” 的首字母缩略词

  • PHP 是一种被广泛使用的开源脚本语言

  • PHP 脚本在服务器上执行

#二、PHP语法

PHP脚本可以放置于文档中的任何位置

PHP脚本以<?php 开头 以 ?>结尾

php的文件扩展名是“.php”；

/php可以包含html标签和一些php的脚本代码/

注释： PHP必须要以分号结尾；

###PHP大小写是敏感的

#三、PHP的变量

变量是储存信息的容器

PHP变量的命名规则

  • 变量以 $ 符号开头，其后是变量的名称
  • 变量名称必须以字母或下划线开头
  • 变量名称不能以数字开头
  • 变量名称只能包含字母数字字符和下划线（A-z、0-9 以及 _）
  • 变量名称对大小写敏感（$y 与 $Y 是两个不同的变量）

创建PHP变量

PHP没有创建变量的命令，变量会在首次为其赋值时被创建；

###PHP是一门松散的语言

我们不必声明PHP 变量的数据类型。PHP 根据它的值，自动把变量转换为正确的数据类型。

###变量的作用域

  • local（局部）
  • global（全局）
  • static（静态）

Local 和 Global 作用域

  • 函数之外声明的变量拥有 Global 作用域，只能在函数以外进行访问。
  • 函数内部声明的变量拥有 LOCAL 作用域，只能在函数内部进行访问。

##前言

说到2d与3d的转换，到底什么是2d，什么又是3d，看下图：

所谓的2d和3d就是我们理解的二维空间和三维空间，那么什么是二维空间，什么又是三维空间呢？

二维空间：二维空间就是一个平面，我们日常生活中所知道的二维坐标系，就是用来解读二维空间的，看图中有x，y，z三个轴，我们只需要看X轴和Y轴，而X轴和Y轴所构成的就叫做二维坐标系，而2d转换就是基于二位坐标系变换的

三维空间：三维空间是基于二维空间上又加了一条Z轴，我们所处的空间就是三维的空间，我们学到的3d转换，就是基于三维坐标系的

当我们弄懂了什么是2d什么又是3d的时候，我们想到什么是变，基于2d和3d又是怎样的变法；

弹性盒子

一、什么叫做flex

Flex是Flexible Box的缩写，意为”弹性布局”，用来为盒状模型设置更为灵活的变化方式；

任何一个容器都可以指定为flex的布局；

.box{ display:block; }

行内元素也可以设置flex的：

.span{ display:inline-flex; }

*注意，设为Flex布局以后，子元素的float、clear和vertical-align属性将失效。

二、flex的概念

设置了flex的元素，称为flex容器（flex container），简称弹性容器。它的所有子元素自动成为容器成员（弹性子项），称为flex项目（flex-item），简称项目

看上图，最大的盒子是弹性容器，里面的三个小盒子是容器的成员，也叫项目或者弹性子项；
容器默认存在两条轴： 水平的主轴（main axis）和垂直的交叉轴（cross axis）。主轴的开始位置叫做main start，结束的位置叫做main end；交叉轴的开始位置叫做cross start，结束的位置叫做cross end；
项目默认沿着主轴排列，单个项目占据的主轴空间叫做main size，占据的交叉轴空间叫做cross size。

三、容器的属性

弹性容器共有六个属性。

1， flex-direction

2，  flex-wrap

3， flex-flow

4，  justify-content

5， align-items

6， align-content

3.1 flex-direction属性

flex-direction属性决定主轴的方向（即项目的排列方向）。

.box{ flex-direction:row | row-reverse | column | column-reverse； }

它有四个值。

1，row（默认值），主轴为水平方向，起点在左端。

2，row-reverse，主轴为水平方向，起点在右端。

3，column：主轴为垂直方向，起点在顶部。

4，column-reverse：主轴为垂直方向，起点在下沿。

3.2 flex-wrap属性

默认情况下，项目都排在一条线（又称”轴线”）上。flex-wrap属性定义，如果一条轴线排不下，如何换行。

.box{ flex-wrap:nowrap | wrap | wrap-reverse; }

它的三个值的表现如下：

（1）nowrap（默认）：不换行。

（2）wrap：换行，第一行在上面。

（3）wrap-reverse:换行，第一行在下面；

3.3 flex-flow

flex-flow属性是flex-direction属性和flex-wrap属性的简写形式，默认值为row nowrap。

.box { flex-flow: <flex-direction> || <flex-wrap>; }

3.4 justify-content属性

justify-content属性定义了项目在主轴上的对齐方式。

.box { justify-content: flex-start | flex-end | center | space-between | space-around; }

假设主轴的方向是在水平位置，从左到右,这五个值的表现形式如下：

flex-start（默认值）：左对齐
flex-end：右对齐
center： 居中
space-between：两端对齐，项目之间的间隔都相等。
space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。

3.5 align-items属性

align-items属性定义项目在交叉轴上如何对齐。

.box{ align-item:flex-start | flex-end | center | baseline | stretch; }

假设交叉轴的方向是从上到下，那么这五个值得具体表现形式如下：

flex-start：交叉轴的起点对齐。
flex-end：交叉轴的终点对齐。
center：交叉轴的中点对齐。
baseline: 项目的第一行文字的基线对齐。
stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。

3.6 align-content属性

align-content属性定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。也就是说弹性子项必须不止一行才会生效；

该属性有6个值：

flex-start：与交叉轴的起点对齐。
flex-end：与交叉轴的终点对齐。
center：与交叉轴的中点对齐。
space-between：与交叉轴两端对齐，轴线之间的间隔平均分布。
space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
stretch（默认值）：轴线占满整个交叉轴。

四、项目的属性

总共有六个属性值：

1， order
2，flex-grow
3，flex-shrink
4，flex-basis
5，flex
6，align-self

4.1 order属性

order属性定义弹性子项的排列顺序。数值越小越靠前，可以为负数，不可以设置小数，默认值为0；

4.2 flex-grow属性

flex-grow属性定义弹性子项的放大比例，默认值为0，即如果存在剩余空间也不放大。

设置这个属性一般都是存在剩余空间，设置的时候，把剩余空间分成n等份，flex-grow：number；把所有的设置这个属性的属性值加起来就是剩余空间，谁的值大谁放大的多，number为几就多出几份

4.3 flex-shrink属性

flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

如果所有项目的flex-shrink属性都为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小，缩小是根据超出容器的差值分配缩小的空间。

负值对该属性无效。

4.4 flex-basis属性

flex-basis属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。

它可以设为跟width或height属性一样的值（比如350px），则项目将占据固定空间。

4.5 flex 属性

flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。

该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。

建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。

4.6 align-self属性

align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

该属性可能取6个值，除了auto，其他都与align-items属性完全一致。
